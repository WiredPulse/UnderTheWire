General Notes:

All of the games are listed on the sidemenu and each are organized in levels. Each level contains a task or challenge and the solution is the password for the following level. All usernames contain the name of the game and the level number (ex: for Century, it would be century1, century2, etc.. and for Trebek, it would be trebek1, trebek2, etc..). All levels have a page on this website and are linked on the sidemenu once you click on a game. You will be using PowerShell to connect to our game server.

It's not expected for anyone to know everything they will encounter in this game, so please don't panic, as the purpose of these games are to learn.

All passwords are lowercase regardless of how they may appear on the screen.

Some things that may help you in the games are below.
          - The Internet
          - Get-Help
          - Get-Command
          - The Tab key will help with finishing out commands
